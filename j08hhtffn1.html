A ranked matrix
Conditions under which the ranked order of matrix elements are invariant.
Applications of the ranked matrix to information.
Consider a set of elements, and a change in their relative quantities.
\(\mu = \left[ {\begin{array}{cc} 1 & 2 \\ 3 & 4 \end{array} } \right]\) 
 
Discrete space.
Consider a discrete space X: {x0, x1, ...}.
 
In terms of the merge operation a discrete space must satisfy the following, which form an alternative definition of a discrete space using the merge operation;
∀i, xi||xi
∀i∀j, i≠j ≡ ¬(xi||xj).
Continuous space.
 
Consider a continuous space X. For N⊂X, let N(x) mean N is a neighbourhood of x.
∀i∀j∃N, (N(xi) ∧ xi||xj) →N(xj).
In a continuous space, x∈N(x), so
(∀i∀j∃N, N(xi) ∧ xi||xj →N(xj)) ≡ (∀i∀j∃N, (xi∈N ∧ xi||xj)  → xj∈N)
Merging space.
In a merging space, the equivalence above does not hold.
¬((∀i∀j,∃N, N(xi) ∧ xi||xj →N(xj)) ≡ (∀i∀j,∃N, xi∈N ∧ xi||xj →xj∈N))
N(xi)∧¬( xi∈N) is not necessarily a contradiction.
 
Set X={{...{xj}...},   xi}
If xi||xj, X={N(xi),  xi}.  
If xj||xj, then X={N( xj), xi}.  
If both xi||xj, xj||xj, then X={N(xj, xi), xi}
If, additionally, xi||xi, then X={N(xj, xi), N(xi), xi}.
